## 📱 Projetos

### 🧠 Alan Turing

Pequeno projeto desenvolvido utilizando **React** para criar uma página apresentando informações sobre **Alan Turing**, considerado um dos pioneiros da computação.

Conteúdos praticados:

* Estrutura básica de um projeto em React
* Componentes
* Organização de interface

---

### 📮 APPCep

Aplicação que utiliza uma **API de consulta de CEP** para buscar informações de endereço a partir de um CEP informado.

Conteúdos praticados:

* Consumo de API
* Requisições HTTP
* Manipulação de dados retornados pela API
* Integração com interface

---

### 📊 Media

Aplicação simples para **cálculo da média de notas de um aluno**, informando se o aluno está **aprovado ou reprovado**.

Conteúdos praticados:

* Lógica de programação
* Estruturas condicionais
* Manipulação de dados de entrada

---

## 🧠 Tecnologias Utilizadas

* **JavaScript**
* **React**
* **APIs externas**
